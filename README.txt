#Edge detection filter

Use [-opr mask] option for setting mask. Possible variables {'sobel','previtt'}
Use [-i path] option for setting input img path.
Use [-o path] option for setting output img path.
Use [-m numberOfThreads] option for setting numberOfThreads. {numberOfThreads > 1}
Use [-r] option for building the performance report with different number of threads. 1 - by default. Set -m value to change it.
